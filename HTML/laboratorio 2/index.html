LOGIN

Bienvenido a "LOGIN". Este es uno de los laboratorios trabajados en una parte del 2do periodo en la clase de Laboratorio 2. Este proyecto funciona como un logeo o como inicio de sesión, en el cual se pueden ingresar diferentes datos, en el cual si alguno de los dos es incorrecto (correo o contraseña), le indique si es incorrecto, y si todo esta correcto, le dirá "Bienvenido".

Datos de prueba:

Correo:"gato@correo.com" Contraseña:"A1234"

Correo:"perro@correo.com" Contraseña:"B1234"

Correo:"tomate@correo.com"Contraseña:"C1234"

Este proyecto se irá mejorando conforme se aprendan nuevas cosas y que sea funcional para el cliente.
